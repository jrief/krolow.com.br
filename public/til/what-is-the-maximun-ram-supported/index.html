What is the maximun ram supported?

Discover how much ram machine supports:

sudo dmidecode -t 16

Output:

  Location: System Board Or Motherboard
  Use: System Memory
  Error Correction Type: None
  Maximum Capacity: 8 GB
  Error Information Handle: Not Provided
  Number Of Devices: 2
